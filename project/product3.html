Nokia 3310 3G

The Nokia 3310 3G is an updated version of the classic Nokia 3310 with 3G network support, allowing faster data connections and better call quality. It kept the nostalgic design while adding modern connectivity.

  • Release Year: 2017
  • Features: 3G connectivity, color display, updated Snake game, FM radio, basic camera
  • Battery Life: Up to 27 days standby time
  • Price: Around $70 at launch
